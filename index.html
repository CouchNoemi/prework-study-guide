HTML

  • HTML meas hypertext markup language.
  • The head element contains information about the page.
  • The p element or paragrahp, contains text or regular content.
  • The heading elements allow you to specify that certain parts of the content are headings or subheadi, this can be from h1 to h6.
  • The body element represent the visible content shown to the user.
  • The Title element sets the title of the page which appears in the browser tab the page is loaded in.
  • The ul tag is used to create an unordered list of items.
  • Inside the ul element we can wrap each list item on a li element.
  • Every HTML element has a opening tag and a closing tag.
  • The void elements have no content.
  • The img element does not wrap content, its purpose is to embed an image.
  • The Source attribute or src contains the path to our image file
  • The alt attribute specify descriptive text for the user that can not see the image
  • The doctype html element is needed to make sure the document behaves correctly.
  • The HTML element wraps all the content on the entire page and is sometimes known as the root element.
  • The meta charset element sets the character from the majority of written languages, it can manage any textual content.

CSS

  • CSS means Cascading style sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A border is the solid line that is just outside the padding
  • A padding indicates how much space we want around the content inside an element.
  • There are multiple types of selectors.
  • The element selector is any HTML element the can be used as a selector.
  • The class selector lets you select HTML elements based on their class attribute, a class in an attribute that all HTML elements can have and is used with css to distinguish elements for specific styling.
  • The id selector it is a HTML attribute but an element can only have 1 id, while it can have multiple classes.
  • The id its got to be unique across the entire web page.
  • Apart from the selector every ruleset must be wrapped in curly braces ({}).
  • within each declaration, you must use a semicolon (;) to separate each declaration from the next one.
  • The properties are ways in which you can style an HTML element, for example the color can be a property of the p element.
  • The property value this is next to the property on the right side after the colon (;),this chooses one out of many possible appearances for a given property.
  • In CSS you can also select multiple elements and apply a single ruleset to all of them, also separate multiple selector by commas.

GIT

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git checkout name of the branch: to change branches.
  • clear: to cleat the terminal.
  • ls: is used to list contents of the current working directory
  • cd: means change directory.
  • git clone: bring a repo down from the internet (remote repository to the local machine.
  • git add: tracks files and changes into git.
  • git commit: save your changes into git.

Javascript

  • Statements are syntax constructs and commands that perform actions.
  • Usually, statements are written on separate lines to make the code more readable
  • JavaScript is a lightweight interpreted or JIT-compiled programming language with first-class functions.
  • In JavaScript, there are 3 types of quotes, double quotes, single quotes, backticks.
