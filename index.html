HTML

  • HTML meas hypertext markup language.
  • The head element contains information about the page.
  • The p element or paragrahp, contains text or regular content.
  • The heading elements allow you to specify that certain parts of the content are headings or subheadi, this can be from h1 to h6.
  • The body element represent the visible content shown to the user.
  • The Title element sets the title of the page which appears in the browser tab the page is loaded in.
  • The ul tag is used to create an unordered list of items.
  • Inside the ul element we can wrap each list item on a li element.
  • Every HTML element has a opening tag and a closing tag.
  • The void elements have no content.
  • The img element does not wrap content, its purpose is to embed an image.
  • The Source attribute or src contains the path to our image file
  • The alt attribute specify descriptive text for the user that can not see the image
  • The doctype html element is needed to make sure the document behaves correctly.
  • The HTML element wraps all the content on the entire page and is sometimes known as the root element.
  • The meta charset element sets the character from the majority of written languages, it can manage any textual content.
